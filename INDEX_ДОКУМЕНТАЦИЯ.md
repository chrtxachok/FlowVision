## 💡 Ключевые понятия проекта

### 🧠 OCR Backend
- **EasyOCR** (основной): Быстрый, надежный, автоматическая загрузка моделей
- **PaddleOCR** (fallback): Точнее, но медленнее, требует конфигурации

### 📋 Документ типы
- **waybill** - Товарная накладная (ТТН)
- **invoice** - Счет-фактура
- **act** - Акт выполненных работ
- **upd** - Уникальный первичный документ

### ⚙️ Обработка
1. **Preprocess** - Подготовка изображения (CLAHE, денойзинг)
2. **OCR** - Распознавание текста (EasyOCR или PaddleOCR)
3. **Postprocess** - Нормализация результатов
4. **Processor** - Извлечение полей (regex patterns)

### 📊 FieldValue
Каждое извлеченное поле содержит:
- `value` - само значение
- `confidence` - уверенность (0.0-1.0)
- `bbox` - координаты на изображении
- `raw_text` - исходный текст из OCR

### ⏱️ Типичное время обработки
- Валидация: 50ms
- Преобработка: 100ms
- **Распознавание: 1000ms** ← самое долгое
- Постобработка: 100ms
- Процессор: 150ms
- **Итого: 2-2.5 сек**

## 🚀 Быстрые команды

### Установка
```bash
git clone https://github.com/your-org/FlowLogix.git
cd FlowLogix
python -m venv venv
source venv/bin/activate  # или venv\Scripts\activate на Windows
pip install -r requirements.txt
```

### Запуск
```bash
uvicorn app.main:app --reload
# http://localhost:8000
```

### Тестирование
```bash
python scripts/test_ocr.py --path static/image.jpg --doc-type waybill
```

```bash

```
