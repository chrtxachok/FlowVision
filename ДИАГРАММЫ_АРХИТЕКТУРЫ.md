## 2. Поток обработки документа

```mermaid
sequenceDiagram
    actor User as 👤 Пользователь
    participant Browser as 🖥️ Браузер
    participant API as 🚀 FastAPI
    participant OCRService as 🧠 OCRService
    participant Pipeline as 📊 OCRPipeline
    participant Processor as ⚙️ Processor
    
    User ->> Browser: 1. Загружает файл
    Browser ->> Browser: 2. Валидирует (тип, размер)
    Browser ->> API: 3. POST /api/v1/ocr/process
    
    API ->> API: 4. Валидирует (API key, content-type)
    API ->> API: 5. Читает file_bytes
    
    API ->> OCRService: 6. process_document(file_bytes, type)
    
    OCRService ->> OCRService: 7. Выбирает backend (EasyOCR/PaddleOCR)
    OCRService ->> Pipeline: 8. bytes_to_numpy() + enhance_image()
    
    Pipeline ->> Pipeline: 9. PIL.open() или PyMuPDF.render()
    Pipeline ->> Pipeline: 10. CLAHE + Binary + Denoise
    
    Pipeline ->> Pipeline: 11. EasyOCR.readtext() ИЛИ PaddleOCR.ocr()
    Pipeline ->> Pipeline: 12. [(bbox, text, confidence), ...]
    
    OCRService ->> OCRService: 13. parse_ocr_result() → normalize
    OCRService ->> OCRService: 14. {"full_text": str, "lines": [...]}
    
    OCRService ->> Processor: 15. WaybillProcessor.process(ocr_result)
    Processor ->> Processor: 16. Regex extraction (5 паттернов на поле)
    Processor ->> Processor: 17. Calculate confidence для каждого поля
    Processor -->> OCRService: 18. {"extracted_data": {...}, "confidence": 0.87}
    
    OCRService -->> API: 19. OCRResponse(status, confidence, data, ...)
    API -->> Browser: 20. 200 OK + JSON response
    
    Browser ->> Browser: 21. displayResults()
    Browser ->> Browser: 22. displayExtractedData()
    Browser ->> User: 23. Показывает результаты ✓
```

## 3. Архитектура OCR Pipeline

```mermaid
graph LR
    Input["🖼️ file_bytes<br/>JPG/PNG/PDF"]
    
    subgraph Preprocess["🔄 PREPROCESSING"]
        P1["Декодирование<br/>PIL.open()"]
        P2["RGB → BGR<br/>cv2.cvtColor()"]
        P3["CLAHE<br/>Контраст"]
        P4["Adaptive<br/>Binary"]
        P5["Морфология<br/>Denoise"]
        P6["Масштабирование<br/>если <500px"]
        
        P1 --> P2 --> P3 --> P4 --> P5 --> P6
    end
    
    subgraph OCRBackend["🧠 OCR ENGINE"]
        Backend{"OCR_BACKEND?"}
        Easy["🟢 EasyOCR<br/>(основной)"]
        Paddle["🟡 PaddleOCR<br/>(fallback)"]
        
        Backend -->|"Try"| Easy
        Backend -->|"if fail"| Paddle
        
        Easy --> EasyRead["Reader.readtext()<br/>CNN + RNN"]
        Paddle --> PaddleOCR["PaddleOCR.ocr()<br/>DB + CRNN"]
    end
    
    subgraph Postprocess["📊 POSTPROCESSING"]
        Parse["parse_ocr_result()"]
        Normalize["Нормализация<br/>в общий формат"]
        Structure["Структурирование<br/>lines + blocks"]
    end
    
    subgraph Output["📤 OUTPUT"]
        Result["OCRResult<br/>{full_text, lines, blocks}"]
    end
    
    Input --> Preprocess
    Preprocess --> P6
    P6 --> OCRBackend
    EasyRead --> Postprocess
    PaddleOCR --> Postprocess
    Parse --> Normalize --> Structure
    Structure --> Output
    
    style Easy fill:#90EE90
    style Paddle fill:#FFD700
```
