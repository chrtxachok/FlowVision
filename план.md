# 🚨 УПРАВЛЕНИЕ РИСКАМИ

| Риск | Индикатор | Действие при реализации |
|------|-----------|------------------------|
| Качество модели ниже целевого | F1 < 0.85 на тесте | Увеличить аугментации, собрать дополнительные данные, скорректировать архитектуру |
| Время обработки >3 секунд | Профилирование показывает узкое место | Оптимизировать препроцессинг, использовать кэширование, рассмотреть квантование |
| Проблемы с развёртыванием в РФ | Ошибки загрузки зависимостей | Использовать локальные зеркала, включить все зависимости в образ |
| Недостаточное покрытие тестами | Покрытие <80% | Добавить тесты для критических сценариев, настроить обязательную проверку в CI |
| Сложности с демонстрацией | Интерфейс не работает на целевом устройстве | Подготовить видео-запись как резерв, протестировать на аналогичном оборудовании |
